【笔记】树莓派配置VNC协议的远程桌面

2020-01-13

前言

树莓派配置VNC协议的远程桌面，实现任意设备通过VNC客户端远程树莓派桌面
MacOS系统可以使用内置的屏幕共享程序远程树莓派桌面

准备工作

  • VNC Viewer

配置 vnc

  • 通过ssh远程连接树莓派

  • 打开树莓派配置

开启 vnc 功能

  • 选择5 Interfacing Options

  • 选择P3 VNC

  • Yes

  • Ok

配置桌面分辨率

  • 选择7 Advanced Options

  • 选择A5 Resolution

  • 选择DMT Mode 82 1920*1080 60Hz 16:9

  • Ok

重启

  • Finish

  • Yes

不重启系统而启动VNC服务（可选）

远程桌面

  • 打开VNC Viewer

  • 通过IP地址连接树莓派

<ip_address>：ip地址
<port>：端口号

  • 输入用户名和密码

用户名：pi
密码：raspberrypi

完成
