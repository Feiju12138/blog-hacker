前言

Java的JUnit单元测试学习笔记

单元测试

  1. 定义一个测试类，类名通常为被测试类名Test，包名通常为被测试包名.test

  2. 定义测试方法，方法名通常为test被测试方法名()，返回值为void，参数列表为空

  3. 添加注解@Junit

  4. 倒入Junit依赖

断言

  • 断言一个结果，与测试的结果做对比，从而判断程序是否有错误，而不是直接输出结果到控制台

<think>：断言的结果
<result>：程序返回的结果

  • 当断言失败时，控制台会打印异常

初始化方法

  • 在所有测试方法执行之前执行该方法

资源释放方法

  • 在所有测试方法执行之后执行该方法
