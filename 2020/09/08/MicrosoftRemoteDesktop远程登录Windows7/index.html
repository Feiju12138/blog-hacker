【笔记】Microsoft Remote Desktop 远程登录 Windows7

2020-09-08

前言

Microsoft Remote Desktop远程登录Windows7

准备工作

  • 非家庭版Windows7系统

远程登录

  • 控制面板->系统和安全->允许远程访问->选择允许远程连接到此计算机

  • 通过选择应用可以设置或查看能登录的用户

  • 控制面板->系统和安全->允许应用通过Windows防火墙->勾选远程协助和远程桌面

  • 通过命令提示符查看本机ip，通过ip地址远程登录

完成
