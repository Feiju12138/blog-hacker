【笔记】UVR5学习笔记

2024-07-21

前言

This application uses state-of-the-art source separation models to remove vocals from audio files. UVR’s core developers trained all of the models provided in this package (except for the Demucs v3 and v4 4-stem models).（Github）
通过UVR5（Ultimate Vocal Remover）实现分离音乐的人声与伴奏

下载依赖

分离音乐的人声与伴奏

  • 批量添加音频文件作为输入

  • 指定输出的目录

  • CHOOSE PROCESS METHOD设置为VR Architecture
  • WINDOW SIZE设置为320
  • ACGRESSION SETTING设置为10
  • CHOOSE VR MODEL设置为后缀为_HP-UVR的选项
  • 勾选GPU Conversion
  • 勾选Instrumental Only表示只提取伴奏
  • 勾选Vocals Only表示只提取人声
  • 点击Start Processing开始分离
