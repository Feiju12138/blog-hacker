【踩坑】npm下载依赖时报错

2024-07-03

前言

npm下载依赖时报错

报错

  • npm error network 'proxy' config is set properly. See: 'npm help config'

原因

  • 配置了网络代理导致无法正常联网

解决问题

  • 配置关闭网络代理

报错

  • 报错：Unexpected end of JSON input while parsing near 'xxx'

原因

  • 之前安装依赖被强制中断了，导致重新安装时安装错误

解决问题

  • 清理缓存

下载全局依赖时报错

解决问题

方法一

  • 按照提示添加权限

方法二

  • 修改下载的全局依赖可执行文件存放位置
