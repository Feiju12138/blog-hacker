【笔记】将WindowsTerminal作为Windows默认终端

2024-08-06

前言

将WindowsTerminal作为Windows默认终端

正文

  • 打开命令提示符，右键标题栏->属性

  • 终端->修改默认终端应用程序改为Windows Terminal->确定

完成

  • 此时在运行时打开cmd会弹出 Windows Terminal 窗口

  • 此时右键开始菜单时，打开Windows Powershell也会弹出 Windows Terminal 窗口
