【笔记】通过Docker部署Homebridge

2024-04-10

前言

通过Docker部署Homebridge

通过DockerCompose部署

初始化

  • 访问http://127.0.0.1:8581

  • GET STARTED

  • 填写用户名和密码->CREATE ACCOUNT

  • OPEN DASHBOARD

集成米家

安装插件

  • 插件->搜索homebridge-miot->选择第一个已认证的插件右下角下载图标

  • 安装

配置插件

  • 插件->Hombridge Miot->配置

  • DISCOVER ALL DEVICES VIA MICLOUD

  • 输入小米用户名和密码->DISCOVER ALL DEVICES

  • 保存

  • 立即重启HOMEBRIDGE

启用插件

  • 插件->Hombridge Miot->配置启用

  • 启用miot

  • 立即重启HOMEBRIDGE

插件配对二维码

  • 插件->Hombridge Miot->配对二维码

  • 苹果家庭APP扫描二维码配对

完成

  • 截图纪念
