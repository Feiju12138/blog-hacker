【笔记】获取 Android Root 权限

2019-12-18

前言

为Lineage OS系统通过刷机的方式获取Root权限

准备工作

  • addonsu-16.0-arm-signed.zip

  • Magisk-v20.1.zip

通过原生方式

刷入 Lineage OS 官方提供的 Root 刷机包

获取 Root 权限

  • 设置->系统

  • 高级

  • 开发者选项

  • Root授权

  • 应用与ADB

管理 Root 权限

  • 管理Root访问

通过 Magisk

刷入 Magisk 来获取 Root 权限

获取 Root 权限

  • 打开Magisk ManagerApp，显示绿色对勾表示已安装

管理 Root 权限

  • Migisk Manager菜单->超级用户
