【笔记】安装WindowsServer2008系统

2021-05-10

前言

安装 Windows Server 2008 （R2） 系统

安装系统

  • 选择语言，直接下一步

  • 现在安装

  • 选择系统版本->下一步

  • 勾选我接受许可协议->下一步

  • 选择自定义（高级）

  • 选择驱动器选项

  • 选择一个未分配空间的磁盘->新建

  • 设定大小->应用

  • 确定

  • 选择刚刚创建的主分区磁盘->下一步

系统初始化

  • 首次登录修改密码->确定

  • 设置管理员密码

  • 您的密码已更改->确定

完成
