【笔记】关闭WindowsServer2008IE增强的安全配置

2021-06-18

前言

关闭Windows Server 2008 IE增强的安全配置以阻止浏览器浏览陌生站点时的安全检测

关闭IE增强的安全配置

  • 右键计算机->管理

  • 服务器管理器->配置 IE ESC

  • 在管理员和用户下都勾选禁用->确定
