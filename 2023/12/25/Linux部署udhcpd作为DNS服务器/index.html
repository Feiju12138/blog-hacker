修改配置

  • 修改配置文件/etc/udhcpd.conf

start、end：IP地址池范围
opt dns：内网DNS服务器

192.168.1.1：本机

option subnet：子网掩码
option dns：公网DNS服务器
option domain：本机域名
option lease：租期，单位秒

864000：10天

  • 修改配置文件/etc/default/udhcpd，注释DHCPD_ENABLED="no"
