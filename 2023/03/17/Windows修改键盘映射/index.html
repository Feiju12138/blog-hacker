【骚操作】Windows修改键盘映射

2023-03-17

前言

Windows修改键盘映射

通过修改注册表永久改变键盘映射

  • 运行regedit打开注册表编辑器

  • 找到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout

  • 空白处右键->新建二进制->名为Scancode Map

第一行16位：全填0
第二行左侧8位：表示接下来将会有几组映射关系，映射关系个数+01即为16进制数填写规则，要求至少有1组映射关系

02：表示接下来将会有1组键盘映射关系
第二行右侧8位：从这8位开始，每8位16进制数表示一组映射关系。每8位中的左侧4位表示
5B E0 3A 00：将原本为大写锁定的按键Caps Lock（16进制数值为00 3A）改变为左侧Alt按键Left Alt（16进制数值为E0 5B）。所有16进制数都要改变每2位的顺序
第三行：假设还有需要映射的按键，继续追加8位16进制数

完成

  • 重启电脑
