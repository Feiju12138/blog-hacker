【笔记】玩客云刷入Armbian
2023-09-10

前言

玩客云（OneCloud）通过线刷的方式刷入Armbian

准备工作

  • Amlogic USB Burning Tool v2.1.6.8
  • Armbian_23.02.0-trunk_Onecloud_jammy_current_6.1.12.burn.img.xz

刷机

  1. USB双公头数据线，一头连接Windows电脑USB口，一头连接靠近HDMI接口的USB口

  2. 打开Amlogic USB Burning Tool烧录工具->文件->导入烧录包

  1. 选择烧录包Armbian_23.02.0-trunk_Onecloud_bullseye_edge_6.2.0_minimal.burn.img.xz->打开

  1. 导入成功后软件底部会显示烧录包文件路径->开始

  1. 首次刷机需要短接，第二次以上可以按住RESET键再上电开机

  2. 等待进度条走完，点击停止

  1. 等待停止完成后，拔出USB数据线，重新上电开机玩客云

首次开机

  • 通过SSH直接连接玩客云或者将玩客云接入键盘都可以，通过默认用户名和密码登录

如果域名无法登录就换成IP地址，具体玩客云的IP地址请查看内网DHCP服务器

username：root
password：1234

  • 修改root用户密码->ctrl+c中断创建新的普通用户

修改时区（可选）

安装docker

安装docker之前切记不要通过apt update && apt upgrade命令更新软件
否则docker安装后会无法启动服务，报错：Cannot connect to the Docker daemon at unix:///var/run/docker.sock. Is the docker daemon running?
如果手动通过systemctl restart docker命令启动服务，会报错：Failed to start Docker Application Container Engine.
通过journalctl -xe命令查看错误日志：The unit docker.socket has entered the 'failed' state with result 'service-start-limit-hit'.
原因不详
