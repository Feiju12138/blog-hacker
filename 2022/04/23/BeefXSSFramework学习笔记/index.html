前言

BeEF is short for The Browser Exploitation Framework. It is a penetration testing tool that focuses on the web browser.（官网）

本文仅用于网络信息防御学习

启动Beef

手动启动

  • 首次启动需要修改默认的用户名和密码

通过Kali自带的脚本启动

手动修改用户名和密码

  • 修改配置文件20行和21行

登录Beef后台

Username：beef
Password：beef（默认密码）
