【笔记】Vim光标移动

2022-02-01

前言

Vim光标移动学习笔记

光标移动

操作 命令模式的快捷键
光标左移一个字符 h
光标下移一个字符 j
光标上移一个字符 k
光标右移一个字符 l

行级移动

操作 命令模式的快捷键 末行模式的快捷键
光标移到当前行行首 0
光标移到当前行行尾 $
光标移到当前行第一个非空字符 ^
光标移动到文件指定行 数字+G :+数字

文件级移动

操作 命令模式的快捷键 末行模式的快捷键
光标移动到文件首 gg :0
光标移动到文件尾 GG :$

单词级移动

操作 命令模式的快捷键
光标移动到上一个单词开头 w、W
光标移动到下一个单词开头 b、B
光标移动到当前单词的末尾 e、E
光标移动到指定单词的下一个单词开头 *
光标移动到指定单词的上一个单词开头 #
光标移动到成对括号的另一个位置 %

段落级移动

操作 命令模式的快捷键
光标移动到当前段落开头 {
光标移动到当前段落末尾 }

屏幕级移动

操作 命令模式的快捷键
光标移动到屏幕第一行 H
光标移动到屏幕最中间一行 M
光标移动到屏幕最后一行 L

翻页

操作 命令模式的快捷键
向上翻页一页 ctrl+f
向下翻页一页 ctrl+b
向上翻页半页 ctrl+u
向下翻页半页 ctrl+d
