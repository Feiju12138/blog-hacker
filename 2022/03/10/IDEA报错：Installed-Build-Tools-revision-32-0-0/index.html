【踩坑】IDEA报错：Installed Build Tools revision 32.0.0...

2022-03-10

前言

IDEA报错：Installed Build Tools revision 32.0.0 is corrupted. Remove and install again using the SDK Manager.

原因

  • AndroidSDK新版本的BUG

解决问题

  • 把/build-tools/32.0.0/文件夹下的d8.bat复制一份，重命名为dx.bat

  • 把/build-tools/32.0.0/lib/文件夹下的d8.jar复制一份，重命名为dx.jar

完成
